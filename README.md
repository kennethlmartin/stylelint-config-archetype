***
**stylelint-config-archetype** - CSS has standards.
***


## Installation

```
npm i --save-dev stylelint-config-archetype
```

## Usage

> Extend configuration to `stylelint.config.js`

```js
module.exports = {
  extends: 'stylelint-config-archetype',
};
```
